# 🎮 Competitive Discord Bot

## 🚀 Setup

### 1. Install dependencies
npm install

### 2. Setup environment
Rename `.env.example` to `.env`

Fill in:
- TOKEN
- CLIENT_ID
- GUILD_ID

### 3. Run bot
npm start

---

## 🧠 Features
- Join queue (GK / DEF / MID / ATT)
- Skill system (1-10)
- Captain draft system
- Pick players one by one
- Start match system
- Auto voice move (Red / Blue VC)
- Auto balance teams
- Final score system
- Rematch system
- Player stats
